<h1>🚧 Proyecto en desarrollo🚧</h1>
Actualmente estoy desarrollando la apliación.

🗒️ Descripción del proyecto
GameJournal es una aplicación que te permite mantener un registro de los juegos a los que vayas jugando. Además, podrás marcar los juegos que no tengas y quieras o los que hayas completado como recomendados. También podrás marcarlos como favoritos!

<h1>❓ ¿Por qué GameJournal? </h1>
La intención de GameJournal es crear una comunidad de jugadores, sin importar tu edad, tu género... 

<h1>✨ Características</h1>
Haz un registro de los juegos que completes.
Descrubre nuevos géneros según tus preferencias.
Añade juegos a tus listas de deseados o favoritos.


<h1>📁 Acceso al proyecto</h1>

Necesitaremos un dispositivo Android o un emulador como el de [Android Studio](https://developer.android.com/studio?hl=es-419)

<h1>🛠️ Abre y ejecuta el proyecto</h1>

Descarga el repositorio en tu dispositivo. 
En caso de estar en un móvil, debes de permitir la instalación de APKs de fuentes externas a la Play Store.
Crea tu cuenta y disfruta de GameJournal, así de sencillo!

El dispositivo en el que se realizó el proyecto es PIXEL 5 usando el API 34. Es recomendable usar este dispositivo virtual y un API igual o mayor a la usada.

Por último, pulsamos en iniciar y la aplicación se iniciará y ya podreis disfrutar de GameJorunal.
